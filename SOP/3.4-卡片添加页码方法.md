# 1 单张修改
* 进入 `浏览 browse` 界面。
* 在 `页码` 字段直接输入 `纯数字`。
    - 正确：1
    - 错误：p1，1-10。

# 2 多张手动修改
## 2.1 依次修改
1. 在当前卡片中输入纯数字。
2. 按 `ctrl + N` 或者 `cmd + N` 进入下一张卡片
3. 然后再输入数字。以此类推。

## 2.2 多张笔记页码替换
1. 选中相同卡片类型、相同页码的笔记。
2. 鼠标点击右键，选择`查找替换`。
3. 查找 = 空
4. 替换 = 页码的纯数字
5. 选择字段 = 页码
6. 确定
